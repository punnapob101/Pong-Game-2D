# 🏓 Pong Game 
**[👉 Play the Game Here! 👈](https://pong-game-2d-atbk29m21-punnapob101s-projects.vercel.app/)**


Welcome to the **Pong Game** with a special local Thai country theme! 🇹🇭✨ 
This is a fun, classic, browser-based Pong game built with simple web technologies that brings you a unique look and playful feel. 🎮💖

---

## 🌟 Description 📝

Challenge yourself in this endless rally! You control the left paddle to play against the right paddle. The game features a charming custom background (`stadium.jpg`) and a beautiful modern UI that makes the experience extra delightful. 🏆🌈

## ✨ Features 🚀

- 🕹️ **Smooth Controls**: Easily control the left paddle using the **Up** and **Down** arrow keys!
- 🖼️ **Cute Visuals**: Enjoy our custom background & aesthetic local vibes. 🌻
- 🌐 **No Install Needed**: Fully built with standard web technologies. Just open and play! 🎉

## 🛠️ Technologies Used 💻

- **HTML5** 🧱: Provides the structure and the magical Canvas element.
- **CSS3** 🎨: Beautiful styling, alignment, flexbox, and background graphics.
- **JavaScript** ⚡: The brain behind the speedy game logic and rendering!

## 🕹️ How to Play 🎯

1. 📂 Open the `Phtml.html` file in any of your favorite web browsers.
2. ⬆️⬇️ Press the **Up Arrow** and **Down Arrow** keys on your keyboard to move your paddle.
3. 🎊 Bounce the ball back and have the best time ever!
